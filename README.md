# 🌤️ Hava Durumu Uygulaması

Gerçek zamanlı hava durumu bilgilerini OpenWeatherMap API'si kullanarak gösteren, sade ve modern arayüzlü bir web uygulaması.

## 🚀 Özellikler

- Şehir adına göre anlık hava durumu sorgulama
- Sıcaklık, hissedilen sıcaklık ve nem bilgisi
- Hava durumu ikonu
- Enter tuşu desteği
- Glassmorphism tasarım

## 🛠️ Kullanılan Teknolojiler

- HTML5
- Tailwind CSS (CDN)
- JavaScript (Fetch API)
- OpenWeatherMap API
